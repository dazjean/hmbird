# mini-next-cli
> mini-next-ssr是一个基于React16+   ReactRouter4.0  koa2.0搭建的一个node服务端渲染框架

# 安装&使用
```
npm install mini-next-cli -g

```

# 初始化mini-next-ssr项目
```
mini-next init yourproject
```

# 更新已有mini-next-ssr项目
> 更新项目会覆盖原始文件，如果对原始文件做过定制，请做好备份！,更新时可新建分支后进行update,然后再和已有项目进行合并
```
cd yourproject
mini-next update
```
